# Agent Selection Guide

## Implementation Tasks

Use:

-free models
- high reasoning effort

In the handoff:

- load `traycer-implement`
- load `traycer-workflow`

The implementation skill is mandatory.

---

## Review Tasks

Use:

-free models
- high reasoning effort

In the handoff:

- load `traycer-review`
- load `traycer-workflow`

The review skill is mandatory.

---

Workspace instructions inside `.traycer/` are additive.
