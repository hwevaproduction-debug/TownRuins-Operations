# Listing Drafts

## Objective
Allow landlords to save incomplete listings.

## Frontend
- save draft button
- draft badge
- resume editing flow

## Backend
- draft status support
- autosave endpoint

## Database
- listing status enum
- updated timestamps

## Edge Cases
- abandoned drafts
- partial uploads
- validation bypassing

## Acceptance Criteria
- drafts save successfully
- drafts are editable
- incomplete listings remain private
